# Titanic Mini Exploratory Data Analysis

## Objective

Perform exploratory data analysis on the Titanic dataset.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Steps

- Data Cleaning
- Missing Value Handling
- Feature Engineering
- GroupBy Analysis
- Histogram
- Heatmap
- Bar Plot

## Key Insights

- Adults survived more than seniors.
- Cherbourg passengers had the highest survival rate.
- Smaller families survived more frequently
